# 教師輔助命題系統 Prompt-only 公開版

本工具是個人維護的教師命題輔助工具，協助老師將教材轉成可複製到 ChatGPT、Gemini 或 Claude 的命題 Prompt。

公開版定位為：

```text
教師命題 Prompt 產生與 AI 回覆整理輔助工具
```

本公開版採 **Prompt-only** 模式，不直接呼叫 AI 後端，也不會自動連線產生試題。repo 不內建 API 金鑰、token、secret、Apps Script URL、backend 或工具程式。

## 使用方式

1. 輸入教材或上傳可讀檔案。
2. 選擇年級、科目與命題模式。
3. 點選「產生命題 Prompt」。
4. 複製 Prompt 到 ChatGPT、Gemini 或 Claude。
5. 將 AI 回覆中的題庫、答案解析與審核表草稿交由教師審查、修正後使用。

## 題庫與審核表流程

題庫模式會產生兩段式命題 Prompt：

- 第一段協助產生評量藍圖與題庫。
- 第二段可依老師選用題號，協助產生組卷配分對帳與試題審核表草稿。

系統可在瀏覽器本機整理 AI 回覆中的固定區塊，方便複製題庫摘要表或審核表草稿；整理結果仍是草稿，需由教師依校內格式修正後使用。

## 支援方向

- 正式考卷 Prompt。
- 一般題庫 Prompt。
- 學力檢測（能力導向）Prompt。
- 國語閱讀理解與文本證據方向。
- AI 回覆固定區塊的本機整理。

## 隱私與安全提醒

請勿輸入或上傳下列內容：

- 學生個資。
- 未公開校務資料。
- 敏感案件內容。
- 不適合公開或交由外部 AI 工具處理的資料。

使用者將 Prompt 複製到外部 AI 工具時，應自行確認該工具的資料使用政策與學校相關規範。

AI 產出內容須由教師審查、修正後使用。本工具不代表學校官方題庫，也不代表正式學力檢測標準。
