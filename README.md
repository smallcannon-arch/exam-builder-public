# 教師輔助命題系統 Prompt-only 公開版

本專案為個人維護之命題輔助工具，公開版採 **Prompt-only** 模式。

本公開版只協助教師依教材、年級、科目與命題模式產生命題 Prompt；系統本身不直接呼叫 AI 後端，也不內建任何 API key、token、secret、Apps Script URL、backend 或工具程式。

目前版本已可直接作為 Prompt-only 公開版使用，並會依教師回饋持續優化。

作者：江志宏｜國小總務主任
聯繫方式：smallcannon@gmail.com

## 定位

- 本工具為教師命題輔助草稿工具。
- 本公開版為 Prompt-only 安全版。
- 本工具為個人維護之教師命題輔助工具，不代表任職學校或任何官方單位立場。
- 本工具不代表任何官方題庫。
- 本工具不代表正式學力檢測標準。
- AI 產出內容須由教師審查、修正後，才可作為教學或評量素材使用。

## 使用方式

1. 輸入教材文字，或上傳系統可讀取的檔案。
2. 選擇年級、科目與命題模式。
3. 產生命題 Prompt。
4. 將 Prompt 複製到 ChatGPT、Gemini 或 Claude 等 AI 工具。
5. 由教師審查、修正、確認後使用。

## 支援方向

- 正式考卷 Prompt
- 一般題庫 Prompt
- 學力檢測（能力導向）Prompt
- 國語閱讀理解與文本證據方向
- 教師快速檢核表

## 隱私與安全提醒

請勿輸入或上傳下列內容：

- 學生個資
- 未公開校務資料
- 敏感案件內容
- 不適合公開或交由外部 AI 工具處理的資料

使用者將 Prompt 複製到外部 AI 工具時，應自行確認該工具的資料使用政策與學校相關規範。

## 本 repo 不包含

- 後端程式
- API key、token、secret
- Apps Script URL
- tools、zip、exe
- 部署文件
- 內部開發文件
- 學校內部行政資料
